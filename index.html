🎯 نظام نقاط بسيط باستخدام كتلة الأوامر🎯 Simple Scoreboard System

/scoreboard objectives add kills totalKillCount
/scoreboard players set @p kills 0
/execute as @e[type=zombie] on death run scoreboard players add @p kills 1

/effect give @p minecraft:speed 30 2
/effect give @p minecraft:strength 20 1
